Database Setup
To get started with this project, follow the steps below to set up the database:

 1. Create Database:

	* Open phpMyAdmin.
	* Create a new database named water_project.

 2. Import Database File:

	* In phpMyAdmin, select the water_project database.
	* Click on the "Import" tab.
	* Choose the SQL file located in the database directory of this project.
	* Click "Go" to import the database file.
